# Traveling Ikea Man Problem

The other day, at a party neary an Ikea, I started discussing the hypothetical problem of needing to travel between every Ikea in the world, only using things from the Ikea. Immediately someone brought up the classical Traveling Salesman Problem - which is the boring and unsolved component of this challenge. Instead this repo will focus on gathering the knowledge necessary to travel from any Ikea to any other Ikea using only things from Ikea.

To clarify, the rules are:

1. You start at an Ikea with nothing.
2. You can only use things that are sold to normal customers (e.g., you cannot use Ikea forklifts).
3. You cannot use things that Ikea customers have on them.
4. You must (eventually) visit every Ikea, and thus travel between them using only equipment from Ikea.
5. All other rules are to be discussed in issues in this repo.
